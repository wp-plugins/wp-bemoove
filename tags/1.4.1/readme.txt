WP-BeMoOvePlugin ～Wordpressで動画を投稿、管理、再生するプラグイン～

== Description ==
WP-BeMoOvePluginはwordpressに動画をアップロードし、
管理し、また再生も行うことのできるプラグインです。
バックエンドには弊社の開発したBeHLSというシステムを用いており
多数の形式の動画に対応しているのが特徴です。

配信方式にはストリーミング配信という、
高セキュリティな方法を採用しております。
PCはもちろん、iOSを中心としたスマートフォンに対応しており、
セキュリティが高いだけでなくマルチデバイス対応にも優れております。

本プラグインは無料で提供しています。
どなたでもご自由に利用いただけます。
プラグインのダウンロード、
インストールをしたら、アカウント作成を行うことで
動画変換機能と配信機能をWordPressの中に組み込むことができます。

また、無料プランではストレージは1GBが上限となっておりますが
10GBの追加プランを有料にてご用意させて頂いております。

[公式ページ](http://www.bemoove.jp/lp/wpplugin/ "Official Site")


== Installation ==
1. 「wp-bemoove」フォルダを「/wp-content/plugins/」ディレクトリにアップロードします。
2. 「プラグインメニュー」からプラグインを起動します。
3. 「BeMoOve」メニューから、アカウントの作成もしくはアカウントを登録して利用開始。
